## 🎯 What is HealthcareWale?

HealthcareWale is an AI-powered SaaS platform that automates healthcare operations for clinics, pharmacies, hospitals, and distributors in semi-urban India — starting with **Uttar Pradesh** and **West Bengal**.

**Core Problem:** Semi-urban Indian healthcare is operationally broken.
- Clinics miss 30–40% of patient calls
- Pharmacies lose ₹3,000–8,000/month to expired medicines
- Pharmacists spend 3 hours/day on manual billing
- Distributors receive orders manually via phone

**Our Solution:** AI voice agents + WhatsApp automation + inventory management — all in one platform, in Hindi & Bengali.

## 🗺️ Roadmap Summary

### Phase 1 — MVP (Month 1–6)
**Theme:** Make 50 pharmacies and 50 clinics never want to leave
- Pharmacy billing + inventory + voice updates + expiry alerts
- Clinic WhatsApp bot + AI voice receptionist + appointment reminders
- Launch in Lucknow, Gorakhpur, Varanasi (UP)

### Phase 2 — Growth (Month 7–18)
**Theme:** Expand to West Bengal. Add Hospital Module.
- Hospital HMS + PMJAY claim filing
- Medicine delivery + lab test booking
- Supplier/distributor module
- West Bengal entry with Swasthya Sathi integration

### Phase 3 — Scale (Month 19–36)
**Theme:** Infrastructure layer for semi-urban Indian healthcare
- Bihar, MP, Jharkhand, Odisha expansion
- White-label for hospital chains + government
- Pharma analytics revenue stream
- ARR ₹10–15 Crore

## 🤖 Tech Stack

```
Frontend:    Next.js 14 + React Native 0.73 + Tailwind CSS
Backend:     Node.js + Fastify + Prisma ORM
Database:    PostgreSQL + Redis + MongoDB + AWS S3
AI/Voice:    Sarvam AI (STT/TTS) + Claude API + Google Vision (OCR)
Automation:  n8n (self-hosted) + BullMQ + Socket.io
WhatsApp:    Interakt BSP (WhatsApp Business API)
Telephony:   Exotel (AI voice calls)
Payments:    Razorpay
Hosting:     AWS Mumbai (ap-south-1)
Govt APIs:   ABDM, NHA (PMJAY), Bhashini, Swasthya Sathi (WB)
```
